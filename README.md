# anjeoni

## 😷 2020 졸업작품 [안전이] App
'안전이’ 앱은 재난 및 사고에 대한 속보와 각 재난의 유형별 행동요령과 안전한 장소로 대피 할 수 있도록 대피소에 대한 정보를 제공하고 사용자 신고 게시판을 통해 신고접수를 제공하는 애플리케이션.

## ⏳ 프로젝트 기간
2019/12 ~ 2020/10
